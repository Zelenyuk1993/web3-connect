Infinity Wallet

Wallet module for connecting Infinity Wallet through web3-connect

Infinity Wallet SDK wallet module for connecting to Web3-Connect. Web3-Connect makes it simple to connect Ethereum hardware and software wallets to your dapp. Features standardized spec compliant web3 providers for all supported wallets, Angular framework, CSS customization, multi-chain and multi-account support, reactive wallet state subscriptions and real-time transaction state change notifications.

Checkout the official Infinity Wallet page for details.
