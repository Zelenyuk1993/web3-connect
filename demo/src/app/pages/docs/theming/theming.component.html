Theming

To customize the color theme of web3-connect and match it with your dApp, you can choose from the available native themes ('dark', 'light', 'system') or create a custom theme using a 'ThemingMap' object.

Available Themes#

To set the color theme of web3-connect to one of the available native themes, import Init from '@b-ee/web3-connect' and pass the theme as a string to the 'theme' init option.

Supported themes:

theme description
'dark' dark mode
'light' light mode
'system' automatically switch between 'dark' & 'light' based on the user's system settings

Variables#

In the table below, you'll find a list of css variables that you can use to theme web3-connect.

variable description
--b-ee-background-color background color
--b-ee-font-family font family
--b-ee-foreground-color foreground color
--b-ee-text-color text color
--b-ee-border-color border color
--b-ee-action-color buttons and links
--b-ee-border-radius border radius

Custom Theme#

To create a custom theme, you can define a 'ThemingMap' object with CSS variables for different components of web3-connect. Pass this object as the theme option.

Dynamically Update Theme with API#

'updateTheme' is an exposed API method for actively updating the theme of web3-connect. The function accepts 'Theme' types (see below). If using the '@b-ee/web3-connect' package there is a hook exposed called 'updateTheme'

The function also accepts a custom built 'ThemingMap' object that contains all or some of the theming variables
